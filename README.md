# WhatsApp Clone

Um clone do WhatsApp Web criado com HTML, CSS e JavaScript puro, replicando a interface e funcionalidades principais do aplicativo.

## 🚀 Funcionalidades

### ✅ Implementadas
- **Interface idêntica ao WhatsApp Web**
- **Lista de contatos** com avatares e status online/offline
- **Chat funcional** com troca de mensagens
- **Pesquisa de contatos** em tempo real
- **Envio de mensagens** com Enter
- **Respostas automáticas** simuladas
- **Status de mensagens** (enviado, entregue, lido)
- **Design responsivo** para mobile e desktop
- **Animações suaves** e transições
- **Cores e tipografia** fiéis ao WhatsApp

### 🎨 Características Visuais
- Cores oficiais do WhatsApp (#25D366, #128C7E, #075E54)
- Layout em duas colunas (contatos + chat)
- Avatares circulares com indicadores de status
- Bolhas de mensagem com cores distintas
- Scrollbars personalizadas
- Ícones Font Awesome

### 📱 Responsividade
- Layout adaptativo para diferentes tamanhos de tela
- Interface otimizada para mobile
- Navegação intuitiva

## 🛠️ Como Usar

1. **Abra o arquivo `index.html`** no seu navegador
2. **Clique nos contatos** na sidebar para trocar de conversa
3. **Digite mensagens** no campo de texto e pressione Enter
4. **Use a pesquisa** para filtrar contatos
5. **Teste os botões** de emoji, anexo e voz

## 🎯 Funcionalidades Interativas

### Chat
- Troca entre diferentes contatos
- Envio de mensagens em tempo real
- Respostas automáticas simuladas
- Atualização automática da última mensagem
- Scroll automático para novas mensagens

### Pesquisa
- Filtro em tempo real por nome ou mensagem
- Interface responsiva durante a busca

### Botões de Ação
- **Emoji**: Adiciona emojis aleatórios
- **Anexo**: Menu simulado de anexos
- **Voz**: Simulação de gravação de voz

## 🎨 Personalização

O site pode ser facilmente personalizado editando:

- **Cores**: Variáveis CSS no arquivo `styles.css`
- **Contatos**: Array `contacts` no arquivo `script.js`
- **Mensagens**: Dados pré-definidos para cada contato
- **Respostas**: Array de respostas automáticas
